# css

Package `css`. CSS-inspired styling system. Handles tokenizing, parsing, and applying CSS-like style rules to widgets. Key responsibilities: tokenizer (`tokenize.go`), parser (`parse.go`), style model (`styles.go`, `model.go`), selector matching (`match.go`), stylesheet management (`stylesheet.go`), scalar values and animations (`scalar.go`, `scalar_animation.go`), and rendering styles to terminal output (`render_styles.go`).
